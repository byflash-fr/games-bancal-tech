⏳ Bancal - Accueil & Instructions

Le jeu se joue de 2 à 20 joueurs dans la même pièce !

🎮 Comment Jouer ?

  • Le Brouillard : Vous ne voyez qu'autour de vous. Vous devez communiquer avec les joueurs dans la pièce pour coopérer !
  • Boutons de groupe : Certains boutons requièrent X joueurs dessus.
  • Formes uniques : Certaines plaques s'activent pour une forme précise (ex: 🔴 Triangle).
  • Objectif : Rejoignez tous la SORTIE avant la fin du temps (5 minutes) !
