🎮 Bancal — La Box

Jeu coopératif local · 2 à 20 joueurs dans la même pièce !

🕹️ Comment jouer ?

  • Brouillard : Vous ne voyez qu'autour de vous — communiquez !
  • Plaques formes : La forme est dessinée sur la plaque. Seul ce joueur peut l'activer !
  • Coop : Certaines plaques demandent X joueurs dessus en même temps
  • Énigmes : Approchez la plaque, répondez A=Oui / B=Non sur votre manette
  • Séquence : Activez les plaques orange dans le bon ordre : 1→2→3
  • Collecte : Ramassez tous les cristaux 💎 éparpillés
  • Sortie : Rejoignez TOUS la sortie pour gagner !
